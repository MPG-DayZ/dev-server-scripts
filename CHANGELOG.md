# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru-RU/1.0.0/).

## 1.3.0

### Добавлено

- **Интерактивный режим выбора пресетов** — удобное консольное меню для выбора пресетов модов и сервера при запуске
  - Навигация стрелками ↑/↓, Home/End, PageUp/PageDown
  - Быстрый выбор по номеру (1-9)
  - Кэширование выбранных пресетов в `.cache/interactive.json`
  - Автоматический выбор при единственном пресете
  - Отмена запуска по Esc или Ctrl+C
  - Включается параметром `"interactive": true` в секции `active` файла `config.json`
  - Параметры командной строки (`-modPreset`, `-serverPreset`) имеют приоритет над интерактивным режимом

### Изменено

- Обновлён `README.md` с документацией по интерактивному режиму
- Добавлен `UPGRADE.md` с инструкцией по добавлению поля `interactive` в существующие конфиги

### Технические детали

- Новый модуль `scripts/interactive.ps1` — переиспользуемый компонент интерактивного меню
- Добавлены строки локализации для интерактивного режима в `scripts/locales.json`
- Кэш хранится в `.cache/interactive.json` с timestamp последнего выбора
